Installation:
------------
1. Download and unpack the Libraries module directory in your modules folder
   (this will usually be "sites/all/modules/").
   Link: http://drupal.org/project/libraries
2. Download and unpack the uc_begateway module directory in your modules folder
   (this will usually be "sites/all/modules/").
3. Download and unpack the beGateway library in "sites/all/libraries".
    Make sure the path to the library file becomes:
    "sites/all/libraries/beGateway"
   Link: https://github.com/beGateway/begateway-api-php
4. Go to "Administer" -> "Modules" and enable the beGateway module.
